## Projectes

### 01. Parklive

Sistema de gestió d'aparcaments basat en microserveis.

Projecte amb dos backends independents, pagaments reals amb Stripe, autenticació social amb Google OAuth 2.0, generació de tiquets en PDF, optimització d'imatges amb Cloudinary i orquestració amb Docker Compose.

Tecnologies: Flask, PHP, Docker, Stripe API, Google OAuth, Cloudinary, MySQL, Nginx, JWT, Bootstrap 5, SASS.

Enllaç: https://github.com/RogerMuntane/Parklive

### 02. Cluedo Web

Joc de misteri web full-stack construït íntegrament en solitari.

Backend Django amb API REST, sistema de sessions nadiu i panell d'admin; frontend Vue 3 amb estètica de dossier policial, cronòmetre real, historial persistent d'intents i estadístiques d'agent per usuari.

Tecnologies: Django, Python, Vue 3, Vite, SQLite, API REST, Django ORM, Bootstrap 5, CSS Scoped.

Enllaç: https://github.com/RogerMuntane

### 03. PathFinder

Plataforma comunitària de viatges amb rutes, votacions, reports i rols diferenciats.

Inclou autenticació de dos factors (2FA), mapes interactius amb Leaflet i OpenStreetMap, cercador amb debounce i filtratge per país i categoria.

Tecnologies: Laravel 12, React 19, TypeScript, Inertia.js, Tailwind CSS, shadcn/ui, Leaflet, Cloudinary, SQLite, 2FA, TipTap.

Enllaç: https://github.com/ErJaLo/PathFinder
